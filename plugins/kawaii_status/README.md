# Kawaii Status 插件

服务器状态查看插件，显示 CPU、内存、磁盘等信息。

## 功能

- 显示服务器 CPU 使用率、频率、核心数
- 显示内存使用情况
- 显示 Swap 使用情况
- 显示磁盘使用情况
- 显示系统信息（CPU 型号、操作系统等）
- 显示框架运行时间
- 显示已加载插件数量

## 使用方法

发送以下命令之一：
- `/status`
- `/状态`
- `/运行状态`

## 配置

在插件配置中可以设置：

- `to_me`: 是否需要在消息中 @Bot 才能触发（默认：false）
- `only_superuser`: 是否只允许超级用户触发（默认：false）
- `nickname`: Bot 昵称（默认：Bot）

## 依赖

- `psutil >= 5.9.8` - 系统信息获取
- `pillow >= 10.3.0` - 图片处理
- `py-cpuinfo >= 9.0.0` - CPU 信息获取

## 安装

将插件目录复制到 `plugins/` 目录下，然后在 Web UI 中启用插件。

## 注意事项

- 插件需要访问系统信息，确保有足够的权限
- 图片生成需要字体文件，请确保资源文件完整
- 首次运行可能需要一些时间来加载字体

## 适配说明

本插件适配自 `nonebot-plugin-kawaii-status`，已适配到 XQNEXT 框架。
