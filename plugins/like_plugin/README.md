# QQ名片点赞插件

适配自 Xiaoyi_QQ 框架的 LikePlugin 插件。

## 功能

- 给用户QQ名片点赞10次
- 每天限制10次
- 支持"赞我"、"超我"、"超湿我"触发词
- 支持查询点赞信息

## 使用方法

### 触发词

- `赞我` - 给发送者点赞10次
- `超我` 或 `超湿我` - 给发送者"超"10次（功能同点赞）
- `{reminder}点赞信息` - 查询今天的点赞信息
- `{reminder}超信息` - 查询今天的"超"信息

### 配置

在插件配置中可以设置：

- `bot_name`: 机器人名称（默认：机器人）
- `reminder`: 触发前缀（默认：空，不需要前缀）

## 数据存储

插件使用框架的存储API保存用户点赞数据，每天自动重置计数。
