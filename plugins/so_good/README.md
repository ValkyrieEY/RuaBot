# SoGood 评价插件

适配自 Xiaoyi_QQ 框架的 SoGood 插件。

## 功能

- 评价用户今天的表现（随机评分 0-100）
- 发电功能（对某人表达内心深处的诉求）
- 支持@用户进行评价或发电

## 使用方法

### 触发词

- `今天棒不棒` 或 `我今天棒不棒` - 评价发送者今天的表现
- `@用户 今天棒不棒` - 评价被@用户今天的表现
- `{reminder}发电` - 对发送者发电
- `{reminder}发电 @用户` - 对被@用户发电
- `{reminder}发电 名字` - 对指定名字发电

### 评分等级

- 0-20: 嗯~今天表现不乖，下次一定要听话哦
- 21-40: 看着顺眼
- 41-60: 亲爱的太棒啦！
- 61-80: 来，抱一个~嗯~
- 81-100: 👍_ _ _👍

### 配置

在插件配置中可以设置：

- `reminder`: 触发前缀（默认：空，不需要前缀）

## 数据存储

插件使用框架的存储API保存用户评分数据和发电词汇。

## 自定义发电词汇

可以通过修改插件存储中的 `words` 数据来自定义发电词汇。词汇中可以使用 `{target_name}` 占位符，会被替换为实际的目标名称。
